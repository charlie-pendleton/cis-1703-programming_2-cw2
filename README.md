How to use:
First Load up the app
Press 2 for GUI or 1 for command line version
Log transactions though the interactive menus
View reports, create budgets and more

Overview:
This Python application helps users manage finances by tracking the income, expenses, while generating a financial report using Object Oriented Programming (OOP). 
It also provides users with the ability to track budgets with alerts and generates financial reports.
This program uses a base transaction class with specialised subclasses demonstrating encapsulation, inheritance and polymorphism. 
This runs using command line interface and a Tkinter GUI, stores data in JSON files, and gurantees a reliable performance.
